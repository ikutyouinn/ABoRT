
# ABoRT Act: Automotive Bill of Rights for Transportation

A model federal bill to protect mobility as a civil liberty, ensure true vehicle ownership/repair rights, mandate efficient infrastructure, and reform transit for safety/self-funding. Inspired by right-to-travel precedents and anti-coercion principles.

## Key Features
- Codifies free travel without checkpoints/surveillance.
- Right to repair, anti-obsolescence, data control.
- Duties on jurisdictions for road maintenance/capacity (tied to federal funds).
- Preemption of conflicting state/local rules.
